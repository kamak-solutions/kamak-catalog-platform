# Kamak Catalog Platform

Plataforma SaaS de catálogo digital para pequenos negócios.

## Visão inicial
Sistema multi-tenant para catálogo de produtos, serviços, eventos e presença digital.

## Stack planejada
- Backend: Node.js + TypeScript + Fastify + Prisma
- Frontend: React
- Banco: PostgreSQL
- Infra: Docker + Nginx
- CI/CD: GitHub Actions
